API 接口文档

以下是小爱音箱管理后台提供的所有 API 接口

基础信息

基础地址: http://localhost:3000

Content-Type: application/json

设备管理

GET /api/device

获取设备信息

请求示例: 
curl http://localhost:3000/api/device
响应示例: 
{
  "success": true,
  "data": {
    "current": { "name": "小爱音箱mini", "miotDID": "89952946" },
    "all": [...]
  }
}

语音播报

POST /api/tts

发送文字让小爱音箱播放

参数:
  • text (string, 必填) - 要播报的文字
请求示例: 
curl -X POST http://localhost:3000/api/tts \
  -H "Content-Type: application/json" \
  -d '{"text": "你好，这是一个测试"}'
响应示例: 
{ "success": true, "data": { "code": 0 } }

唤醒小爱

POST /api/wake

远程唤醒小爱音箱，让它进入聆听状态

请求示例: 
curl -X POST http://localhost:3000/api/wake
响应示例: 
{ "success": true, "data": { "code": 0 } }

音乐播放

POST /api/music/play

搜索并播放音乐

参数:
  • keyword (string, 可选) - 搜索关键词
请求示例: 
curl -X POST http://localhost:3000/api/music/play \
  -H "Content-Type: application/json" \
  -d '{"keyword": "周杰伦 晴天"}'
响应示例: 
{ "success": true, "data": { "code": 0 } }

音频播放

POST /api/audio/play

播放指定的音频文件 URL

参数:
  • url (string, 必填) - 音频文件 URL（支持 MP3、WAV 等格式）
请求示例: 
curl -X POST http://localhost:3000/api/audio/play \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/music.mp3"}'
响应示例: 
{ "success": true, "data": { "code": 0 } }

🎙️ 智能广播

POST /api/ai/control

通过自然语言创建智能广播，AI 会根据需求生成合适的广播内容

功能：
  • 📢 即时广播 - 立即播放生成的广播内容
  • ⏰ 定时广播 - 创建定时广播任务
  • 🎵 音频播放 - 播放音频库的音频
请求示例: 
# 即时广播
curl -X POST http://localhost:3000/api/ai/control \
  -H "Content-Type: application/json" \
  -d '{"command": "播报今天天气晴朗25度"}'

# 定时广播
curl -X POST http://localhost:3000/api/ai/control \
  -H "Content-Type: application/json" \
  -d '{"command": "明早7点叫我起床"}'

# 播放音频
curl -X POST http://localhost:3000/api/ai/control \
  -H "Content-Type: application/json" \
  -d '{"command": "播放起床铃声"}'

🤖 AI 对话

POST /api/ai

调用大模型进行问答，支持上下文记忆

参数:
  • question (string, 必填) - 用户问题
请求示例: 
curl -X POST http://localhost:3000/api/ai \
  -H "Content-Type: application/json" \
  -d '{"question": "地球为什么是圆的？"}'
响应示例: 
{
  "success": true,
  "data": {
    "question": "地球为什么是圆的？",
    "answer": "地球之所以是圆的，是因为..."
  }
}

定时任务

GET /api/schedules

获取所有定时任务列表

请求示例: 
curl http://localhost:3000/api/schedules
POST /api/schedules

添加新的定时任务

参数:
  • text (string, 必填) - 消息内容
  • type (string, 必填) - 类型: cron/interval/once
  • cron (string) - Cron 表达式 (type=cron)
  • interval (number) - 间隔秒数 (type=interval)
  • at (string) - 执行时间 (type=once)
请求示例 (Cron): 
curl -X POST http://localhost:3000/api/schedules \
  -H "Content-Type: application/json" \
  -d '{"text": "早上好", "type": "cron", "cron": "0 8 * * *"}'
请求示例 (间隔): 
curl -X POST http://localhost:3000/api/schedules \
  -H "Content-Type: application/json" \
  -d '{"text": "该喝水了", "type": "interval", "interval": 3600}'
DELETE /api/schedules/:id

删除指定的定时任务

请求示例: 
curl -X DELETE http://localhost:3000/api/schedules/abc123
